What is Docker

Docker is the company driving operating-system-level virtualization known as containerization, through a container platform. containerization allows companies to address every application across an increasingly diverse portfolio of clouds. With containerization, Dockers allowes better collaboration and innovation in business. Source: https://www.docker.com/what-docker, https://www.google.com/search?q=what+is+docker&oq=what+is+docker&aqs=chrome.0.0l6.4448j0j9&sourceid=chrome&ie=UTF-8

What is Nginx

Nginx is a free, open-source, HTTP and reverse proxy server, as well as a mail proxy server. Nginx is known for its high performance, stability and low resource consumption, among other things. Source: https://www.nginx.com/resources/wiki/, https://nginx.org/en/

What is VM (virtual machine)

VM is a software program or operating system that behaves like an actual computer; also capable of performing tasks such as running applications and programs like a separate computer. In other words, creating a computer within a computer. Source: https://www.techopedia.com/definition/4805/virtual-machine-vm, https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/

Who is Cloud Computing Service Provider - Digitalocean

DigitalOcean is a cloud computing vendor that offers Infrastructure as a Service (IaaS) for software developers. Through a private Virtual Machine (VM), called "droplet." Source: https://searchcloudcomputing.techtarget.com/definition/DigitalOcean
